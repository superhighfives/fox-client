# Fox (client)

The client for the Brightly Fox music video.

## Requirements

Install yeoman:

    npm install -g yo

Download the audio:

    mkdir app/media
    cd app/media
    curl -O "https://s3-ap-southeast-2.amazonaws.com/fox.wearebrightly.com/media/fox.{mp3,ogg}"

## Running

    grunt server
